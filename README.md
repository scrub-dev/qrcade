# ![QRCade](./assets/qrcade.png)
## Version 2.0.0 pre-release
A simple QR Code-based mobile exercise gaming platform designed to provide an augmented gaming platform based in the real world.

### The Stack:
Backend:
  * Typescript
  * Express
  * Sequelize (Sqlite)

Frontend:
  * Typescript
  * React
  * Vite
  * Tailwind
